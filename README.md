# Projeto de Monitoramento de Frequência - Frequency

## 📌 Sobre o Projeto

Este projeto consiste em um sistema web desenvolvido para o controle e monitoramento da frequência de alunos em ambiente escolar. A aplicação permite que professores e gestores registrem presenças, acompanhem o desempenho dos estudantes e gerenciem informações acadêmicas de forma prática e organizada.

---

## 🎯 Objetivo

Facilitar o controle de frequência escolar, oferecendo:

* Registro rápido de presenças e faltas;
* Visualização de dados dos alunos;
* Acompanhamento individual;
* Apoio à tomada de decisões pedagógicas.

---

## ⚙️ Funcionalidades

* Cadastro de alunos
* Listagem de alunos
* Registro de frequência
* Relatório individual do aluno
* Painel com resumo de dados
* Sistema de login e controle de acesso
* Gerenciamento de usuários

---

## 🛠️ Tecnologias Utilizadas

* **HTML** – Estrutura das páginas
* **CSS** – Estilização
* **PHP** – Lógica e backend
* **MySQL** – Banco de dados
* **JavaScript** *(opcional)* – Interatividade

---

## 🗂️ Estrutura do Sistema

O sistema é composto por:

* Painel principal (dashboard)
* Lista de alunos
* Tela de frequência
* Relatório individual do aluno
* Painel de usuários
* Sistema de autenticação
